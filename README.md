## 🚀 Quick Start

### 1. Use this template

Click **"Use this template"** at the top of this repo, or clone it:

```bash
git clone https://github.com/bloxiebuilds/myfoliokit-Portfolio-Template.git
cd myfoliokit-Portfolio-Template
```

### 2. Open `index.html` and edit your info

Everything you need to change is clearly marked with `<!-- Edit me! -->` comments:

- Your **name** in the nav and hero
- Your **bio** in the About section
- Your **skills** tags
- Your **project cards** (icon, name, description, link)
- Your **email** and **social links**

### 3. Deploy to GitHub Pages (free!)

1. Push your repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site is live at `https://YOUR-GITHUB-USERNAME.github.io/myfoliokit-Portfolio-Template` 🎉

## 🎨 Customization

### Change the accent color

In `index.html`, find this line near the top:

```css
--accent: #6EE7B7;
```

Replace `#6EE7B7` with any color you like!

### Add a project card

Find the `<!-- PROJECT CARD -->` comment and duplicate the block:

```html
<a href="YOUR_LINK" class="project-card">
  <div class="project-icon">🔥</div>
  <h3>Project Name</h3>
  <p>Short description of the project.</p>
  <span class="project-tag">View Project →</span>
</a>
```

### Add a skill tag

```html
<span class="skill-tag">Your Skill</span>
```
